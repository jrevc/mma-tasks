A1: Project Brief

For your Final Project you can either do your CAPSTONE PROJECT, or a NEW PROJECT:

Capstone Project

  1. Members of a capstone project can share in the assets and design of a project; HOWEVER the output should be individual. This means each member will develop the website independently from the other. When your teacher checks the website, each member will not have the same HTML/CSS.

New Project

  1. This can be anything except a personal portfolio.
  2. It should be a new website; it cannot be a version of an existing website (for example, you cannot redesign the website of Lady Gaga.)
  3. Here are some examples of interesting projects:
    • small business of a friend or family
    • tutorials (makeup, art, some craft)
    • informational (taking care of hamsters, dressing up for different occasions, etc.)
    • advocacy (reducing plastics, how we can help reduce global warming, etc.)
  4. The content must be substantial. Consult your teacher on this.

Submission

  1. Before doing or submitting this assignment, check the assessment rubrics associated with it so you can see how you will be graded. (You'll be able to view this on BigSky once the dropbox is accessible)
  2. Download the Template MMINTER PROJECT SHEET. Accomplish it. Then submit it as a PDF file. (Do not save the PDF for high-resolution printing because it will end with a very large file size. Your teacher only needs the "screen" resolution (about 72ppi to 100ppi) so save it in this resolution so that the file size is 10MB or smaller. The smaller the better with acceptable image quality.)
